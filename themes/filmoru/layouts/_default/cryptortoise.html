How It Works

▼

▼

▼

▼

▼

▼

▼

▼

Advanced: Build your own key

Cryptortoise is open source. If you want to source and flash your own hardware:

  1. Get a Raspberry Pi Pico (~$5)
  2. Flash Pico FIDO firmware (UF2)
  3. Install Cryptortoise from GitHub

Any FIDO2 key with hmac-secret support works (YubiKey 5, SoloKeys, Nitrokey).
